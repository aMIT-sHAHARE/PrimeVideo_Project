# 📊 Amazon Prime Video Data Analysis Dashboard (Power BI)

## 🔍 Overview

This project is an end-to-end data analysis of the Amazon Prime Video dataset using Power BI. It transforms raw streaming data into meaningful business insights through interactive dashboards, KPI tracking, and visual storytelling.

The dashboard highlights content distribution, genre trends, ratings classification, country-wise availability, and release year growth patterns.

---

## 🎯 Objectives

- Analyze distribution of Movies vs TV Shows  
- Identify top-performing genres  
- Study content growth over time  
- Understand audience ratings classification  
- Explore country-wise content availability  
- Build an interactive business dashboard  

---

## 📁 Dataset

- **Dataset Name:** Amazon Prime Titles  
- **Format:** CSV  
- **Records:** 9,655 Titles  
- **Includes:**  
  - Title Name  
  - Genre  
  - Release Year  
  - Rating  
  - Country  
  - Duration  
  - Director  
  - Type (Movie / TV Show)

# 📊 Prime Video Dashboard Insights

## 📌 Executive Summary

This dashboard analyzes Prime Video's content library from **1920 to 2021** using 9,655 titles.  
The platform shows strong movie dominance, rapid recent expansion, and high demand for drama-driven content.

---

# 🔢 Key Statistics

| Metric | Value |
|-------|------|
| Total Titles | 9,655 |
| Rating Categories | 25 |
| Genres | 519 |
| Directors | 5,771 |
| Timeline | 1920 - 2021 |

---

# 🎬 Content Type Analysis

| Content Type | Titles | Share |
|-------------|-------|------|
| Movies | 7,810 | 80.82% |
| TV Shows | 1,850 | 19.18% |

### 📌 Insight:
Prime Video’s library is heavily movie-focused.  
TV Shows represent a smaller share, creating an opportunity for stronger retention through episodic content.

---

# 👥 Audience Rating Analysis

| Rating | Titles |
|------|------|
| 13+ | 2,117 |
| 16+ | 1,547 |
| ALL | 1,268 |
| 18+ | 1,243 |
| R | 1,010 |

### 📌 Insight:
Most titles target teenagers and adults, indicating focus on mainstream entertainment audiences.

---

# 🎭 Top Genre Analysis

| Genre | Titles |
|------|------|
| Drama | 986 |
| Comedy | 536 |
| Drama, Suspense | 399 |
| Comedy, Drama | 377 |
| Animation, Kids | 356 |
| Documentary | 350 |

### 📌 Insight:
Drama is the leading genre with nearly **2x more titles than Comedy**, showing strong audience demand for story-driven content.

---

# 🌍 Country-wise Analysis

Strong content presence observed in:

- United States  
- India  
- United Kingdom  
- Japan  

### 📌 Insight:
Prime Video has a global footprint with strong positioning in major entertainment markets.

---

# 📈 Release Year Trend

- Limited content before 2000  
- Growth accelerated after 2010  
- Peak additions during 2018–2021  

### 📌 Insight:
Prime Video aggressively scaled its content library in the last decade to compete in the streaming market.

---

# 💡 Business Recommendations

- Increase TV Show share for better retention  
- Expand high-performing genres like Drama & Comedy  
- Invest in regional/local language content  
- Grow family-friendly categories  
- Continue original content strategy
